Lenguajes de programación en Adalab

En el módulo 1 aprenderemos todo lo que se necesita para crear una página web: conceptos básicos de HTML y CSS, diseño web responsive utilizando Flexbox y Grid. La base de una página web es el HTML y el CSS, con HTML se escribe el contenido de la página y con CSS se modifica el aspecto que va a tener. Al final del módulo 1 sabremos maquetar con HTML y CSS una página web profesional desde cero.

En el módulo 2 aprenderemos Javascript y, con ello, a programar. Programar es dar órdenes al ordenador para conseguir un resultado a partir de unos datos previos. Para trabajar con el navegador lo que haremos será aprender a darle esas órdenes en código. Javascript es un lenguaje de programación que permite programar páginas web y aplicaciones y darle interacción a las mismas.
Al finalizar este módulo sabremos hacer una página web interactiva con Javascript(JS), que responda a las acciones del usuario y esto significa que las páginas web que hagas por ejemplo podrán tener formularios, alertas, mensajes, que los botones y enlaces tengan animaciones y datos obtenidos de bases de datos.

En el módulo 3 aprenderemos React.js. React JS es JavaScript con vitaminas. Por ello en este módulo vamos a afianzar las bases de JavaScript a la vez que vamos a aprender este framework. React JS junta HTML, CSS y Javascript para que podamos hacer las aplicaciones separadas por componentes y que sea extensible y reutilizable, haremos aplicaciones web muy limpias y ordenadas, es decir, vamos a trabajar como se hace en los proyectos reales.
